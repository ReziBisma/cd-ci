# CI/CD Final Project

## Project Name

Simple CI/CD Pipeline Application

## Description

This project demonstrates the implementation of a complete CI/CD pipeline using:

* GitHub Actions for CI automation
* Tekton for pipeline orchestration
* OpenShift for application deployment

## Features

* Automated build process
* Continuous integration with GitHub Actions
* Pipeline execution using Tekton
* Deployment to OpenShift cluster
